\subsection{Установка ПО}\label{stlinkinstall}

Для работы с чипами STM32 используя программаторы STlink, встроенные на демоплаты Discovery
необходимо установить пакет STSW-LINK004 (STM32 ST-link Utility и драйвер), скачав его с
\url{http://www.st.com/web/en/catalog/tools/PF258168}

Подключаете отладочную плату, запускаете \stlink\ Utility 
\menu{Пуск>Программы>STMicroelectrinics>STM32 ST-LINK Utility}

\bigskip
\includegraphics[width=\textwidth]{adapter/stlink/install/5.jpg}
\bigskip

Запускаем \menu{ST-LINK>Firmware update>Device Connect} и проверяем что прошивка \stlink
не устарела (ПК должен быть подключен к Сети), при необходимости нажимаем \keys{Yes}
и обновляемся.
