Для начала освоения программирования для \arm\ рекомендуем использовать бесплатный пакет от Keil:
\url{http://www.keil.com/arm/mdk.asp} --- ограничения бесплатной версии в 32К кода вполне достаточно
для начального освоения программирования под процессоры семейства \cortex{Mx}, а затем уже
можно переползать на открытое ПО: GNU toolchain \ref{gcc}, Eclipse \ref{eclipse} и Linux \ref{linux}.

Процесс установки и первоначальной настройки описан в \ref{keilinstall}.
В этом разделе будут подробно описаны возможности и настройки среды.
