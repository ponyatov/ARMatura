\subsection{Настройка глобальных свойств}

Глобальные свойдства настраиваются в меню \menu{WIndows>Preferences}.

\bigskip

Прежде всего настроим автосохранение при запуске \verb+Build+ \keys{Ctrl+B}:

\begin{itemize}
  \item запуск сборки при сохранении файла
  \item автоматическое сохранение всех измененных файлов перед сборкой
  \item автосохранение рабочих файлов
  \item кодировка файлов с кириллицей (может потребоваться корректировка при импорте
  файлов в кодировках KOI8, UTF8)
  \item конец строки в стиле dos/windows: \verb+<CR><LF>+ (в UNIX используется только \verb+<LF>+)
\end{itemize}
