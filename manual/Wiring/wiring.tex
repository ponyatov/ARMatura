\bigskip
\wiring --- OpenSource фреймворк для программирования микроконтроллеров.

\bigskip

\wiringпозволяет писать кросс-платформенный код для управления устйроствами, подключенными к 
широкому диапазоны различный микрокнтроллерных плат, для создания разнообразных интерактивных 
объектов, среативных пространств или физических экспериментов. Этот фреймворк thoughtfully
дизайнерами и художниками предполагая создание сообщества где начинающие с помощью экспертов
со всего мира обмениваются идеями, знаниями и коллективным опытом. Сущесвуют тысячи студентов, 
художников, дизайнеров, исследователей и хоббитов, которые используют \wiring\ (или его частную 
реализацию Arduino) для исследований, прототипирования, и производства законченных устройств.

\begin{itemize}
  \item Roadmap включает поддержку различных аппаратных ядер\footnote{на самом деле только AVR8}
  \item The current AVR8 Core supports the Wiring hardware and any hardware based on the AVR 
    atmega processors. AVR Xmega, AVR Tiny, TI MSP430, Microchip PIC24/32 Series and STM M3 
    ARM Cores will be available soon\footnote{судя по активности визуалдизайнеров --- году к 2048}.
\end{itemize}

На текущий момент существует одна единственная мощная реализация: Arduino, на базе 8-битных
микроконтроллеров Atmel AVR8 (ATmega). Попытки реализации \wiring\ для других МК выливаются
в замкнутые на себя микросообщества, собирающиеся вокруг единственного производителя
железа: например \href{http://leaflabs.com/devices/}{LeafLabs Maple} (пара плат на базе 
\ref{STM32F103CBT}), дорогой \href{http://arduino.cc/en/Main/ArduinoBoardDue}{Arduino Due},
ардуклоны на MSP430\etc

\bigskip
Для \armatura\ мы пойдем тем же путем --- сделаем минимальную реализацию \wiring\ и наплодим
кучку несовместимых библиотек. Желающие могут присоединяться к разработке.
