\chapter{\armatura\ /STM32F407IGT/} \label{armatura}

Большая полнофункциональная плата с самым мощным из доступных MCU
STM32F407IGT\ref{STM32F407IGT}, набором наплатной периферии, которая чаще всего нужна
для автономного контроллера, и коннектором шины \bus\ для подключения модулей расширения.

\bigskip

\begin{itemize}
  \item MCU \cortex{M4F}: 1M Flash, 192K SRAM, LQFP144, 123 GPIO
  \item FPGA
  \item $2\times$USB
  \item $2\times$CAN
  \item SDIO
  \item \bus
  \item 4x16 LCD
  \item joystick
  \item $4\times$LED
  \item Audio in (jack 3.5) с поддержкой микрофона
  \item Audio out (jack 3.5) со стереоDAC и усилителем
  \item коннектор Dallas 1wire
  \item 6-осевой акселерометр
  \item поддержка \robo
\end{itemize}
